# Car-Accident-Dashboard
This Tableau dashboard displays accident statistics broken down by vehicle type. It also shows accident trends and a map showing the geographical location of incidents.
## üîç Key features

- **Comparison of periods**: Detailed comparison of indicators for the current (`current_year`) and previous (`previous_year`) years
- **Severity analysis**: Filtering and grouping data by the `Accident Severity` parameter
- **Transport analytics**: Breakdown of accidents by vehicle type (`Vehicle type`)
- **Dynamic trends**: Visualization of changes in key indicators over time
- **Factor analysis**: A study of the influence of weather conditions and types of road surface on accidents
